Q10829: L-Gap Substrings
如果一個 string 符合 UVU 的形式，其中 U 不是空的，V 有剛好 L 個字元，我們就說 UVU 是 L-Gap string。例如 abcbabc 是 1-Gap string。xyxyxyxyxy 是 2-Gap string 也是 6-Gap string，但是不是 10-Gap string(因為 U 不能是空的)。

給你一個string s，和一個正整數 g，你要找出 s 裡面有多少 g-Gap substring。s 只包含小寫字母，最多 50,000 個字元。

Input
第一行有一個整數 t(1<=t<=10)，測資的數量。接下來的 t 行每行都有一個整數 g 和一個 string s。
Output
對每組測資，輸出測資號碼和 g-Gap substring 的數量。細節請看範例輸出。
Sample Input Sample Output 
2
1 bbaabaaaaa
5 abxxxxxab
 
Case 1: 7
Case 2: 1
